# MyStudio IDE

An IDE built from scratch in Rust as a learning experience.

### Prerequisites

1. [Rust](https://rust-lang.org) (min v1.56)
2. GTK3+
3. gtk-rs (0.15+)

### Get Started

1. Clone this repository.

2. `cargo run`

> The project will compile and you should see the IDE.
